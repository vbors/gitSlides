Workflow Commands - Checkout

git checkout [branch] - change local branch, if there are local changes, it will try to merge
git checkout -b [branchName] [start-point] - Create new local branch, referencing to start-point branch or tag

Example: git checkout -b jump_2_8_0_feature origin/jump_2_8_0 will create a new local branch from remote branch jump_2_8_0

Not to be confused with CVS checkout, in git this command it's called clone
