Installation of Git support into Eclipse

  • Install plugin from repository: Select the Help ▸ Install new Software…​ menu entry.
  • Set Work with: http://download.eclipse.org/egit/updates and select all items.

The Eclipse workspace and Git repositories

It is good practice to place your Git repositories outside the Eclipse workspace. This separates your Git repository from any additional meta-data which Eclipse might create. By default, Eclipse Git uses the git folder in the users home directory to clone new repositories
