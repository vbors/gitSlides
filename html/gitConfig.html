Configurations

Git store the configuration into gitconfig files

  • /etc/gitconfig file: Contains values for every user on the system and all their repositories --system
  • ~/.gitconfig or ~/.config/git/config file: Specific to your user. --global
  • .git/config file: Repository youâ€™re currently using: Specific to that single repository

Set your Identity

$ git config --global user.name "Vasile Bors"

$ git config --global user.email vasile.bors@kronsoft.eu

Checking Your Settings

$ git config --list
